打开注册表编辑器

首先打开注册表编辑器，可以通过 Win+R 快捷键打开运行，然后输入 regedit 打开，也可以直接在任务栏的搜索框搜索“注册表编辑器”

添加所需的项

文件夹中空白处的右键菜单

然后依次展开 Background\shell，在 shell 处右键，点击 新建-项 ，新建项的名字可以随意，我这里为了方便识别写 AndroidStudio ，选中新建的 AndroidStudio 项，右侧会有一个叫”(默认)”的数据项，双击”(默认)”，填入 Open Folder as AndroidStudio Project ，这个就是最终在右键菜单中显示的内容，你也可以填写其他内容比如 在 Android Studio 中打开 。

然后在左侧 AndroidStudio 这一项点右键，点击 新建-字符串值，新建值的名字写 Icon ，双击 Icon ，在数值数据内填入 Android Studio 的安装路径，具体到exe文件，注意如果是64位系统这里要选 studio64.exe 。

接下来还是在左侧 AndroidStdio 的地方右键，点击 新建-项，新建项的名字填 command ，双击右侧的 “(默认)”，在数值数据内填入如下格式的内容：
